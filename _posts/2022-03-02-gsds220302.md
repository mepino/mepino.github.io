1. 리눅스 강의
- 리눅스 "~" 표시는 home 디렉토리에 있다는 뜻
- cd / : root 디렉토리로 이동
- cd ~ : user home 디렉토리로 이동
- pwd : 현재위치
- 절대경로 : root 디렉토리 기준으로 위치 표시
- cat, echo, rm, mv [old] [new], cp [old] [new], mkdir [dir], rmdir [dir], man [command]
